# JAVA 数据类型

## 基本数据类型

- 整型

  - **byte （范围：-128 到 127）**
  - **short (范围：-32768 到 32767)**
  - **int (范围：-2147483648 到 2147483647)**
  - **long (范围：-9,223,372,036,854,775,808 到 -9,223,372,036,854,775,808)**

- 浮点型

  - **float (单精度)**
  - **double (双精度)**

- 布尔型 

  > 只有`true` 和`false`

  - **true**
  - **false**

- 字符型

  - **char 只能是单个字符,支持中文**
